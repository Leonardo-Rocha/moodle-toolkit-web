Overview

O moodle toolkit é uma ferramenta criada para facilitar a conversão de arquivos PDF para os dois principais formatos acadêmicos: TeX e GIFT.

TeX é referência mundial para a escrita de artigos científicos e é o padrão utilizado nas grandes revistas e conferências.

GIFT é uma extensão do Markdown, utilizado pelo Moodle para gerar os questionários online.

Pensando nisso, foi criada uma forma de que com base em um questionário PDF (como por exemplo as provas do ENADE), fosse gerado um arquivo de texto, que deve ser manualmente formatado em markdown e automaticamente convertido nos formatos desejados.

Para mais instruções, veja a página de exemplos.
